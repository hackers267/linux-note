# Samba

Samba 是一种灵活的文件共享工具，它将你可能在组织中运行的所有不同平台结合在一起。

## 在Linux上安装Samba

在  *ArchLinux* 上安装  *Samba* 只需要执行下面的命令就行：

```
  paru -S samba
```
